# 🎲 Mystery Shop Roulette

A single-page website with one button that sends you to a random product listing on AliExpress. No algorithm, no recommendations — just chaos.

**[Try it →](https://your-username.github.io/your-repo/)**

## How it works

AliExpress product URLs follow the pattern `aliexpress.com/item/{ID}.html`. The page generates a random ID from known product ID ranges and opens it in a new tab. Not every ID resolves to a live listing — that's part of the fun.
